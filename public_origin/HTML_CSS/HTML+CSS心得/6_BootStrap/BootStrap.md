# BootStrap简介

* Bootstrap 是全球最受欢迎的前端开源工具库，它支持 Sass 变量和 mixin、响应式栅格系统、自带大量组件和众多强大的 JavaScript 插件。基于 Bootstrap 提供的强大功能，能够让你快速设计并定制你的网站。

* Bootstrap 是全球最受欢迎的前端框架，用于构建 **响应式、移动设备** 优先的网站。
